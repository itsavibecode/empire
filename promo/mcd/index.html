Set where it sits, how big it is, and what today's day number should be. The URL below updates as you change things — copy it when you're happy.

2 · Install as a browser source

Works in OBS Studio, Streamlabs Desktop, and anything else that supports a Browser Source. The background is transparent — it layers cleanly over your scene.

  1. Add a Browser Source

    In your streaming software, open your scene, click + under Sources, and choose Browser (OBS) or Browser Source (Streamlabs). Name it Sobriety Overlay.

  2. Paste the URL

    Paste the URL you copied above into the URL field. Leave custom CSS blank.

  3. Set the canvas size

    Set Width: 500 and Height: 400. That's enough room for the achievement + counter. Check Shutdown source when not visible off so the counter keeps ticking between scenes.

  4. Position in your scene

    Click OK, then drag the source to the corner of your canvas. The overlay anchors itself inside its own box — resize the source if you need it bigger or smaller.

  5. Day rolls over automatically

    The counter ticks up at your local midnight. No editing needed — leave the browser source running and it'll read 13 tomorrow, 14 the day after, and so on.

  6. Need to reset the count?

    Come back here, change Today reads… to whatever it should be right now, copy the new URL, and paste it into the Browser Source properties. Done.

3 · URL parameters (advanced)

For anyone who'd rather edit the URL by hand.

ParameterValuesWhat it does
pos tl · tr · bl · br Which corner of the browser-source box the overlay anchors to. Default tl.
days integer, e.g. 12 Sets today's count. The overlay then auto-increments from there at local midnight.
start YYYY-MM-DD Explicit sobriety start date. Overrides days if both are set.
scale decimal, e.g. 1.2 Overall size multiplier. 1 is default.
