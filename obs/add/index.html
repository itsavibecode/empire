Setup

Three steps. One browser source. One minute.

The BRB screen is a single URL you add to OBS (or Streamlabs, or any other broadcaster) as a browser source. Set it once, bind it to a hotkey, and you've got a professional BRB that promotes the whole network every time you step away.

  1. Create a new scene for the BRB screen

    In the Scenes panel (bottom-left), click + and name it BRB. Keep your existing scenes untouched — this one is just for when you step away.

  2. Add a Browser source

    With the new BRB scene selected, click + in the Sources panel and choose Browser. Name it Empire X BRB and click OK.

  3. Paste the URL and set the size

    Paste https://ourempirex.com/obs/ into the URL field. Set Width to 1920 and Height to 1080. Leave Custom CSS empty.

  4. Check two important boxes

    In the same source properties window, scroll down and enable both:

    • Shutdown source when not visible — saves CPU while you're live on your main scene.
    • Refresh browser when scene becomes active — re-pulls the latest live streamers every time you switch to BRB.

    Click OK to save.

  5. Assign a hotkey (optional but strongly recommended)

    Go to File → Settings → Hotkeys, scroll to your BRB scene, and bind a key to Switch to scene. Something like F9 or a Stream Deck button is perfect — one tap and you're on BRB instantly.

  6. Done. Test it.

    Hit your hotkey or click the BRB scene. You should see the full-screen Empire X overlay with live Kick embeds of whoever's streaming right now. When you come back, just switch to your main scene and the BRB source shuts down automatically.

  1. Any browser-source-capable encoder works

    The BRB overlay is just a URL. If your streaming software can load a web page as a source, it'll work. This includes Kick Studio, Twitch Studio, XSplit, and most IRL encoders with "browser layer" support.

  2. Add a browser / web source

    Find the "Add Source" or "Add Layer" option in your encoder. Choose Browser, Web Source, or HTML. Paste https://ourempirex.com/obs/.

  3. Set the canvas to 1920 × 1080

    The overlay is designed for a 1920 × 1080 canvas. If your encoder only supports scaling (not canvas resolution), scale the browser source to fill the screen.

  4. Assign to a scene / layer

    Put the source on its own dedicated scene/layer so you can switch to it instantly without disturbing your main layout.

Troubleshooting

Common questions, handled.

The overlay is loading but no streamers show up
If nobody from the Confirmed Streamers list is live on Kick right now, the overlay shows a "WARMING UP" placeholder instead of an empty grid. As soon as anyone goes live, they'll populate automatically — the overlay re-checks Kick every 90 seconds.
One streamer is live but not appearing in the grid
The overlay uses public CORS proxies (allorigins.win, corsproxy.io) to check Kick's API. Occasionally a single request gets rate-limited and silently drops. The overlay re-checks every 90 seconds, so the streamer will appear on the next cycle. If you see the same name missing for 5+ minutes, their Kick handle in the roster may not match their actual username exactly — check kick.com/{handle} loads their page.
The featured streamer's audio isn't playing
Browser autoplay policy blocks audio from embedded players until there's a user gesture. In OBS, right-click the Empire X BRB browser source → Interact → click anywhere in the preview window. Audio will play from then on. Do this once per OBS session. For a permanent fix, launch OBS with the command-line flag --autoplay-policy=no-user-gesture-required.
The background video isn't playing
Refresh the browser source (right-click → Refresh). OBS's browser engine sometimes blocks video autoplay on first load. If it still doesn't play after a refresh, check your internet connection — the overlay loads videos from the /obs/videos/ directory, so they need to be reachable.
Can I add my own stream to the featured grid?
The grid pulls from the Confirmed Streamers list on ourempirex.com. To get added, DM @our.empire on Instagram — once you're listed, you'll appear in the BRB whenever you go live on Kick.
Does this cost anything? Any API keys needed?
No. The overlay is free, hosted on the Empire X site, and doesn't need any credentials. Just drop the URL into OBS and go.
Will this slow down my stream?
With Shutdown source when not visible enabled (step 4), the browser source uses zero CPU while you're on your main scene. It only spins up when you switch to the BRB scene. Load is minimal — a handful of Kick embeds and a looping background video. Even on modest hardware it runs fine.
Can I customize the look?
The hosted version at ourempirex.com/obs/ stays consistent for the whole network. If you want a custom version (different colors, your logo, different rotation time), you can fork the source files — reach out on Instagram and we'll point you to the repo.
